About

I use p5.js for creative coding — generative art, interactive simulations, and visual experiments. I also build data maps using Leaflet and public datasets from my work at WorkForce WV and school projects at Marshall University.

All data stays local to this repository — no external database required. Each visualization is either embedded here directly or linked to the p5.js Editor / an external page.

Add a Visualization

Edit assets/visualizations.json to add new entries. Each entry supports:

  • type: "p5js" or "map"
  • url: link or embeddable iframe URL
  • embed: true to show an inline iframe toggle
  • thumbnail: path to a preview image in assets/gallery/
